# 🍕 The Great Pizza Catastrophe

A spooky and humorous **text-based adventure game** built with Python where the player makes choices that determine how the story ends.

---

## 📖 Project Description

**The Great Pizza Catastrophe** is an interactive command-line adventure game.
The player takes the role of a pizza delivery driver sent to deliver pizza to a mysterious and spooky mansion.

When the player arrives, they meet a pale man named **Steve** and notice strange things around the mansion, including:

* A crying werewolf
* A glowing box labeled **DO NOT OPEN**
* A bat-shaped pet door

The player must make decisions that lead to different outcomes. Some choices allow the player to escape safely, while others lead to funny or disastrous endings.

The game uses conditional logic and user input to create multiple story paths.

---

## 🎮 Game Features

* Interactive **player choices**
* Multiple story branches
* Several unique endings
* Restart option to replay the game
* Spooky and humorous storytelling
* Case-insensitive input handling

---

## 🛠 Technologies Used

**Language**

* Python

**Concepts Used**

* `input()` for player interaction
* `print()` for displaying story text
* `if`, `elif`, and `else` for decision making
* `while` loops for restarting the game
* `.lower()` to handle user input consistently
* Multi-line strings (`""" """`) for story narration
